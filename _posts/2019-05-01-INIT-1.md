## Code Block Examples

### Powershell
```powershell
$colors = @("Red","Orange","Yellow","Green","Blue","Indigo","Violet")
For ($x=0; $x -lt $colors.Length; $x++) {
    $colors[$x]
}
```

### Python
```python
colors = ["Red","Orange","Yellow","Green","Blue","Indigo","Violet"]
for x in colors:
  print(x)
```

### Javascript
