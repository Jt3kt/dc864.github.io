## Code Block Examples

### Bash
```bash
declare -a colors=("Red" "Orange" "Yellow" "Green" "Blue" "Indigo" "Violet")
for x in "${colors[@]}"
do
   echo "$x"
done
```

### Javascript
```javascript
var colors = {"Red","Orange","Yellow","Green","Blue","Indigo","Violet"}; 
for (x = 0; x < colors.length; i++) { 
  text += colors[x] + "<br>";
}
```
